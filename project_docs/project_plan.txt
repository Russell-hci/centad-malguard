# Recommended Final Project Direction

## Proposed Research Theme

> Robust and Efficient Malware Classification Using Lightweight Deep Learning Models Under Adversarial Attacks

This is much stronger than:

> “malware detection using CNNs”

because it introduces:

* robustness,
* efficiency,
* practical deployment,
* and adversarial cybersecurity evaluation.

That immediately sounds more research-oriented.

---

# Core Research Question

Your entire project should revolve around ONE central question:

> Can lightweight malware image classifiers remain accurate and robust under adversarial attack conditions while still being efficient enough for deployment on resource-constrained devices?

This gives your whole project a clean identity.

# Biggest Risks (And Mitigation)

## Risk 1 — Dataset Too Small

Mitigation:

* use multiple datasets,
* augmentation,
* transfer learning.

---

## Risk 2 — Training Instability

Mitigation:

* start with pretrained models,
* freeze layers initially.

---

## Risk 3 — Scope Explosion

Mitigation:

* prioritize:

  1. baseline models,
  2. attacks,
  3. one defense method,
  4. Grad-CAM.

Everything else is optional.
